Welcome to the **Student Productivity Management System (SPMS)**! This is a complete, beginner-friendly desktop application built entirely in Java using Swing. It is designed to act as an all-in-one productivity assistant for university students to track their studies, manage their schedules, and calculate their grades.

---

## ✨ Features

- **📊 Modern Dashboard**: A beautiful, real-time command center displaying total productivity points, pending tasks, and study sessions.
- **🔐 Secure Login System**: Persistent user accounts saving data securely across sessions.
- **⏱️ Study Focus Tracker**: A customizable study timer that rewards you with live productivity points for staying focused! Tracks your complete session history.
- **✅ Smart Task Manager**: Add, complete, and delete tasks. Automatically sorts and suggests tasks based on priority. Completing tasks earns you productivity points!
- **🧮 Advanced GPA Calculator**: Calculate complex Semester GPAs including combined Theory & Lab credits (strictly enforcing Sessional caps) and maintain a running **Cumulative CGPA**.
- **📅 Dynamic Timetable**: A day-wise schedule filter that automatically detects the current day. Easily add, edit, or delete classes without conflicts!
- **📂 Rich File Organizer**: Import PDFs and files by subject, and create your own **Rich-Text Notes** directly in the app. Full CRUD capabilities including physical file deletion.

---

## 🛠️ Technologies Used

- **Language**: Java 
- **UI Framework**: Java Swing & AWT (Custom Graphics2D styling for rounded cards and modern UI)
- **Build Tool**: Maven
- **Architecture**: Object-Oriented Programming (OOP) core concepts including Encapsulation, Inheritance, and Polymorphism.
- **Persistence**: Local Text-based Database (`.txt`) using a custom `StorageManager`.

## 🚀 Getting Started

### Prerequisites
Make sure you have [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html) installed and your `JAVA_HOME` environment variable configured.

### Running the App
1. Clone the repository:
   ```bash
   git clone https://github.com/CH-Hanan/SPMS.git
   ```
2. Navigate into the directory:
   ```bash
   cd SPMS
   ```
3. Run the application from your IDE by executing the `Main.java` file, or via Maven.

*(Note: The default credentials are **Username:** `admin` | **Password:** `admin`)*
